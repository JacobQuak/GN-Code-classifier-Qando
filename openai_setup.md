# 🔐 OpenAI API Key Aanvragen en Instellen

OpenaAI modellen worden gebruikt om de AI nodes binnen de workflow te runnen. Je hebt hiervoor een persoonlijke **OpenAI API key** nodig.

---

## 🧠 Waarom een OpenAI API Key?

De classifier maakt gebruik van OpenAI’s GPT-model om codefragmenten automatisch te classificeren. Om dit te laten werken, is een geldige sleutel vereist voor authenticatie bij de OpenAI API.

---

## ✅ Stap 1 – Log in op het Qando OpenAI-account

1. Ga naar: [https://platform.openai.com/account/api-keys](https://platform.openai.com/account/api-keys)
2. Log in met de Qando OpenAI inloggegevens  


---

## ✅ Stap 2 – Vraag een nieuwe API key aan

1. Ga naar: [https://platform.openai.com/account/api-keys](https://platform.openai.com/account/api-keys)
2. Klik op **“Create new secret key”**
3. Geef eventueel een herkenbare naam op (bijv. `n8n-code-classifier`)
4. Kopieer de gegenereerde API key direct – **je kunt deze later niet opnieuw bekijken**

> 🔒 Deel deze key niet. Hij is persoonlijk en geeft toegang tot je gebruikslimiet.
